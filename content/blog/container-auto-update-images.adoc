:source-language: shell

We can set the desired image pull policy during container startup.

.Podman/docker
----
$ podman/docker container run --pull=always
$ podman/docker container run --pull=newer
----


[source, yaml]
.Kubernetes
----
spec:
  containers:
    - name: <name>
      imagePullPolicy: Always
----
